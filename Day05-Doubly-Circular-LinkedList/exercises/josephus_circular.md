# Josephus Problem (Circular List)

## Problem
Given n people in a circle, eliminate every k-th person until one remains. Return the survivor's position.

## Input
- Integer n
- Integer k

## Output
- Survivor position

## Constraints
- 1 <= n <= 10^5
- 1 <= k <= 10^5

## Hint
- Simulate with a circular list or use the recurrence formula.
