# Recursive Fibonacci

## Problem
Compute the nth Fibonacci number using recursion.

## Input
- A single integer n (0 <= n <= 40)

## Output
- The nth Fibonacci number.

## Constraints
- Keep n small to avoid exponential runtime.

## Hint
- Base cases: F(0) = 0, F(1) = 1
- Recursive case: F(n) = F(n-1) + F(n-2)
