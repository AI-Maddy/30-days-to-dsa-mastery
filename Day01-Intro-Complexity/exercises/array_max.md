# Maximum Element in an Array

## Problem
Given an array of integers, return the maximum element.

## Input
- An integer n (size of array)
- n integers

## Output
- The maximum value in the array.

## Constraints
- 1 <= n <= 10^5
- Each element fits in 32-bit signed integer

## Hint
- Keep a variable for the max seen so far and update while scanning.
