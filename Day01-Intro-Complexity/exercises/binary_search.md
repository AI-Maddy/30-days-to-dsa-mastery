# Binary Search

## Problem
Given a sorted array and a target value, return the index of the target or -1 if not found.

## Input
- An integer n (size of array)
- n sorted integers (ascending)
- An integer target

## Output
- Index of target or -1

## Constraints
- 1 <= n <= 10^6

## Hint
- Use two pointers and shrink the search range based on mid.
